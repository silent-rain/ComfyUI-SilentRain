# 项目搭建

## 安装

```shell
cargo install maturin
```

## 初始化项目

```shell
maturin init
```

## 构建python包

```shell
maturin build
```

## 部署到当前环境

```shell
maturin develop
```

## 发布到pypi

```shell
maturin publish
```

## 上传python包到pypi

```shell
maturin upload
```
